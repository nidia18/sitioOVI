DISEÑO DE SITIOS WEB (UNAD)

QUE ES EL OVI

OVI Es un recurso digital el cual está diseñado para suministrar información a los usuarios e interactuar con ellos. Gracias a esto se puede compartir noticias, compartidas a través de páginas web, blog, redes sociales, correos, entre otros.

OBJETIVO DEL OVI

Es un objeto virtual. - Se pueden crear actividades on line - Utilizan lo mas relevante para el aprendizaje incluido: textos, imágenes, videos, animación, preguntas y ejercicios - La metodología de aprendizaje es dirigida CONTENIDOS DIGITALES Pueden ser cualquier forma de información u datos digitales entre ellos gráficos, fotos, videos, documentos de investigación, artículos, informes, estadísticas, e-book, música etc.

LA VENTAJA DEL OVI

La ventaja del uso de este recurso digital es que la información que se brinda a los usuarios se puede compartir a través de páginas Web, Correos, Redes Sociales, revistas digitales, blogs etc. OVA Son objetos virtuales de aprendizaje los cuales se enfocan en el desarrollo de actividades para definir u evaluar competencias, niveles educativos etc.

OBJETIVOS

El curso diseño de sitios web se divide en tres unidades, las cuales son: 1. Introduccion al diseño de sitios web, en esta vemos que es y como funciona el sistema de control de verciones GIT y el uso de GITHUB para sincronizar nuestros proyectos a travez de internet y asi poder trabajar de forma colaborativa, 2. HTML5 y 3. CSS3. En las unidades 2 y 3 veremos el uso de HTML y CSS para desarrollar paginas web.

El OVI Diseño de sitios web esta elaborado con el fin de dar a conocer informacion basica sobre el manejo de herramientas enfocadas al desarrollo web como los son GIT, GITHUB, HTML5 Y CSS3. Esto se realizara a travez de lecturas, videos y actividades cortas y faciles de entender. El objetivo de este OVI es el de dar a conocer a los visitantes del sitio informacion basica sobre que es y como funciona GIT, GITHUB, HTML5 y CSS3, ademas de presentar unos tutoriales sobre su intalacion y uso.
